# Soli - Sistema de Gestão Empresarial

## Sobre o projeto

Soli é um sistema de gestão empresarial desenvolvido com React, TypeScript e FastAPI.

## Tecnologias

- **Frontend**: Vite, React, TypeScript, shadcn/ui, Tailwind CSS
- **Backend**: FastAPI, PostgreSQL

## Como executar

```sh
# Instalar dependências
cd frontend_app_soli && npm install

# Iniciar servidor de desenvolvimento
npm run dev
```

O backend deve estar a correr em `http://127.0.0.1:8000`.

## Scripts

- `npm run dev` - Servidor de desenvolvimento
- `npm run build` - Build de produção
- `npm run preview` - Pré-visualizar build
- `npm run lint` - Executar ESLint
